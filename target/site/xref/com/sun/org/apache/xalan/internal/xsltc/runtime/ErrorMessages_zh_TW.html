33  /*
34   * XSLTC run-time error messages.
35   *
36   * General notes to translators and definitions:
37   *
38   *   1) XSLTC is the name of the product.  It is an acronym for XML Stylesheet:
39   *      Transformations Compiler
40   *
41   *   2) A stylesheet is a description of how to transform an input XML document
42   *      into a resultant output XML document (or HTML document or text)
43   *
44   *   3) An axis is a particular "dimension" in a tree representation of an XML
45   *      document; the nodes in the tree are divided along different axes.
46   *      Traversing the "child" axis, for instance, means that the program
47   *      would visit each child of a particular node; traversing the "descendant"
48   *      axis means that the program would visit the child nodes of a particular
49   *      node, their children, and so on until the leaf nodes of the tree are
50   *      reached.
51   *
52   *   4) An iterator is an object that traverses nodes in a tree along a
53   *      particular axis, one at a time.
54   *
55   *   5) An element is a mark-up tag in an XML document; an attribute is a
56   *      modifier on the tag.  For example, in <elem attr='val' attr2='val2'>
57   *      "elem" is an element name, "attr" and "attr2" are attribute names with
58   *      the values "val" and "val2", respectively.
59   *
60   *   6) A namespace declaration is a special attribute that is used to associate
61   *      a prefix with a URI (the namespace).  The meanings of element names and
62   *      attribute names that use that prefix are defined with respect to that
63   *      namespace.
64   *
65   *   7) DOM is an acronym for Document Object Model.  It is a tree
66   *      representation of an XML document.
67   *
68   *      SAX is an acronym for the Simple API for XML processing.  It is an API
69   *      used inform an XML processor (in this case XSLTC) of the structure and
70   *      content of an XML document.
71   *
72   *      Input to the stylesheet processor can come from an XML parser in the
73   *      form of a DOM tree or through the SAX API.
74   *
75   *   8) DTD is a document type declaration.  It is a way of specifying the
76   *      grammar for an XML file, the names and types of elements, attributes,
77   *      etc.
78   *
79   *   9) Translet is an invented term that refers to the class file that contains
80   *      the compiled form of a stylesheet.
81   */
