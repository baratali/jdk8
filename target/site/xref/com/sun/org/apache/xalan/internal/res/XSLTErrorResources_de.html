43  /*
44   * This file contains error and warning messages related to Xalan Error
45   * Handling.
46   *
47   *  General notes to translators:
48   *
49   *  1) Xalan (or more properly, Xalan-interpretive) and XSLTC are names of
50   *     components.
51   *     XSLT is an acronym for "XML Stylesheet Language: Transformations".
52   *     XSLTC is an acronym for XSLT Compiler.
53   *
54   *  2) A stylesheet is a description of how to transform an input XML document
55   *     into a resultant XML document (or HTML document or text).  The
56   *     stylesheet itself is described in the form of an XML document.
57   *
58   *  3) A template is a component of a stylesheet that is used to match a
59   *     particular portion of an input document and specifies the form of the
60   *     corresponding portion of the output document.
61   *
62   *  4) An element is a mark-up tag in an XML document; an attribute is a
63   *     modifier on the tag.  For example, in <elem attr='val' attr2='val2'>
64   *     "elem" is an element name, "attr" and "attr2" are attribute names with
65   *     the values "val" and "val2", respectively.
66   *
67   *  5) A namespace declaration is a special attribute that is used to associate
68   *     a prefix with a URI (the namespace).  The meanings of element names and
69   *     attribute names that use that prefix are defined with respect to that
70   *     namespace.
71   *
72   *  6) "Translet" is an invented term that describes the class file that
73   *     results from compiling an XML stylesheet into a Java class.
74   *
75   *  7) XPath is a specification that describes a notation for identifying
76   *     nodes in a tree-structured representation of an XML document.  An
77   *     instance of that notation is referred to as an XPath expression.
78   *
79   */
