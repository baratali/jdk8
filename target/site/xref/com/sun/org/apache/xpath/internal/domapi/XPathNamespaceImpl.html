38  /**
39   *
40   *
41   * The <code>XPathNamespace</code> interface is returned by
42   * <code>XPathResult</code> interfaces to represent the XPath namespace node
43   * type that DOM lacks. There is no public constructor for this node type.
44   * Attempts to place it into a hierarchy or a NamedNodeMap result in a
45   * <code>DOMException</code> with the code <code>HIERARCHY_REQUEST_ERR</code>
46   * . This node is read only, so methods or setting of attributes that would
47   * mutate the node result in a DOMException with the code
48   * <code>NO_MODIFICATION_ALLOWED_ERR</code>.
49   * <p>The core specification describes attributes of the <code>Node</code>
50   * interface that are different for different node node types but does not
51   * describe <code>XPATH_NAMESPACE_NODE</code>, so here is a description of
52   * those attributes for this node type. All attributes of <code>Node</code>
53   * not described in this section have a <code>null</code> or
54   * <code>false</code> value.
55   * <p><code>ownerDocument</code> matches the <code>ownerDocument</code> of the
56   * <code>ownerElement</code> even if the element is later adopted.
57   * <p><code>prefix</code> is the prefix of the namespace represented by the
58   * node.
59   * <p><code>nodeName</code> is the same as <code>prefix</code>.
60   * <p><code>nodeType</code> is equal to <code>XPATH_NAMESPACE_NODE</code>.
61   * <p><code>namespaceURI</code> is the namespace URI of the namespace
62   * represented by the node.
63   * <p><code>adoptNode</code>, <code>cloneNode</code>, and
64   * <code>importNode</code> fail on this node type by raising a
65   * <code>DOMException</code> with the code <code>NOT_SUPPORTED_ERR</code>.In
66   * future versions of the XPath specification, the definition of a namespace
67   * node may be changed incomatibly, in which case incompatible changes to
68   * field values may be required to implement versions beyond XPath 1.0.
69   * <p>See also the <a href='http://www.w3.org/TR/2004/NOTE-DOM-Level-3-XPath-20040226'>Document Object Model (DOM) Level 3 XPath Specification</a>.
70   *
71   * This implementation wraps the DOM attribute node that contained the
72   * namespace declaration.
73   * @xsl.usage internal
74   */
